# Muhammad Abdullah — Developer Portfolio

A modern, fully responsive personal portfolio website built with **HTML, CSS, Bootstrap 5, and JavaScript**.

🔗 **Live site:** _add your GitHub Pages link here once deployed_

## ✨ Features

- Premium dark theme with a custom cyan → violet → pink gradient
- Light / dark mode toggle (remembers your choice)
- Fully responsive — mobile, tablet, and desktop
- Smooth scroll, scroll-reveal animations, animated counters & skill bars
- Working contact form (Formspree backend)
- Downloadable CV (`cv.html` → PDF)
- Custom favicon

## 📂 Sections

Hero · About · Skills · Projects · Services · Contact · Footer

## 🛠️ Built with

HTML5 · CSS3 · Bootstrap 5 · JavaScript · Bootstrap Icons · Google Fonts

## 🚀 Run locally

```bash
python -m http.server 8000
```

Then open <http://localhost:8000>.
